# Plans 文档说明

本目录保存跨 UI 与工程的产品阶段、路线图、范围边界、迁移策略和关键产品决策。

## 写作边界

- Roadmap 文档说明“当前阶段做什么、不做什么、后续待设计什么”。
- 稳定 UI 设计指导放入 `../ui/`。
- 稳定工程实现规则放入 `../engineering/`。
- 文件名不使用 `[done]`、`[active]` 等状态前缀；状态写在文档正文中。

## 当前文档

- `planned-training-roadmap.md`：计划训练（Planned Training）路线图。
- `training-metrics-roadmap.md`：训练指标（Training Metrics）路线图。
