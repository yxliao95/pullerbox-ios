# 待规划范围

本文档记录尚未进入稳定根目录说明的版本范围、后续计划和待设计内容。

## 用户流程待设计

- 自由训练。
- 新记录列表。
- 记录详情。
- 历史趋势和统计入口。
- 训练中跳过当前阶段。
- 训练中修改计划或动作参数。
- App 被系统终止后的训练恢复。
- 中途连接设备并从计时模式（timerOnly）切换到拉力计模式（forceDevice）。

## 训练指标待设计

- 更多力量和疲劳指标。
- 更细粒度的趋势对比。
- 个性化阈值。
- 训练内容变化后的趋势合并策略。
- 力竭信号在训练中的实时提示方式。

## 设计待细化

- 记录详情页的信息层级、曲线标注密度和指标解释方式。
- 历史趋势页的入口、筛选、图表类型和空状态。
- 多动作、多指标对比时的交互结构。
